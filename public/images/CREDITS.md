## Replacing with client photography

These are demo placeholders. To use the client's own professional photography,
replace each file in place (keep the same filename), ideally at ~1600px wide or
larger. Then regenerate the blur placeholders in `src/lib/images.ts` if desired.
No code changes are required to swap the images themselves.
